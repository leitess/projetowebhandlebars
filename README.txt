Sistema Web
===========

Projeto desenvolvido para a disciplina: laboratorio de desenvolvimento multiplataforma. Onde
o objetivo será cadastrado possiveis agendamentos de serviços.

Este projeto está usando as bibliotecas: Express e Handlebars. Para instalar use o comando:
      npm install

Para rodar use o comando:
      npm run dev

As rotas são:
    GET: Para cadastrar, a endpoint é / -> http://localhost:8081/
    GET: Para listar, a endpoint é /consultar -> http://localhost:8081/consultar
    GET: Parar atualizar, a endpoint é /atualizar -> http://localhost:8081/atualizar
    - / - index.handlebars
- /consultar - consultar.handlebars
- /atualizar - atualizar.handlebars
